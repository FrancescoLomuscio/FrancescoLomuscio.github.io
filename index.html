Procedural Map System

Tools
Unity C#
Ownership
UI Infrastructure Textures
Product
Unpublished
Platforms
Microsoft

Design and development of a procedural map generation system for an in-progress extraction shooter, focused on creating coherent and navigable environments from structured data inputs. I was responsible for the full design and implementation of the system. The core requirement was to generate maps starting from multiple sets of points, which would then be connected through a functional road network. To achieve this, I implemented a Delaunay triangulation algorithm to generate an initial graph, followed by a pruning phase to remove unnecessary edges and define a cleaner, more intentional layout. From this graph, I generated the road network and corresponding mesh. On top of this structure, I designed and implemented a grid-based generation system. Each cell is assigned a type based on its distance and direction relative to the nearest road, allowing for context-aware placement of assets and procedural floor texturing. This approach ensures spatial coherence while maintaining variety across generated maps.
